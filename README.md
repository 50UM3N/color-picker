# Installation

This application made using [ReactJs](https://reactjs.org/) and [Tauri](https://tauri.studio/).

## Dependencies

-   [Node Js](https://nodejs.org/en/)

### Installing Dependencies

    npm install

### Build the client

    npm build

### Build the Application

    npm run tauri build
